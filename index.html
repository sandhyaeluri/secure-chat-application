⭐ Key Features
  • 🔒 End-to-end encryption — every message encrypted on sender's device using Fernet (AES-128-CBC) before transmission
  • 🔑 RSA-2048 key exchange — session key shared securely using asymmetric encryption, preventing MITM attacks
  • 👤 Secure authentication — password-based user registration and login with MySQL database storage
  • 🗄️ MySQL database with 3 normalised tables — signup, login, and chat rooms with public/private room support
  • 🛡️ Security hardened — protected against SQL injection and XSS attacks through input validation and sanitisation
  • 📊 Functional testing — 8 documented test cases covering signup, login, room creation and encryption with 100% pass rate on positive scenarios
